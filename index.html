shaped-0.1.0.0: Mutable and immutable arrays with multi-dimensional indexing

Provides multi-dimentional arrays by adding a shape to a vector from the vector package using indicies from the linear package. Supports mutable a immutable arrays with a similar interface to vector but with a strong lens flavour.
